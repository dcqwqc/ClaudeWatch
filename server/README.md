# Claude Watch Server Setup

This directory contains the server-side components for the Claude Watch app.

## Quick Installation & Configuration

For the best experience, use the interactive installer. It will download the latest scripts to `~/.claude-watch` and guide you through the entire setup process, including Firebase, dependencies, and hooks.

**On your server**, run this command:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/dcqwqc/ClaudeWatch/main/server/install.sh)"
```
